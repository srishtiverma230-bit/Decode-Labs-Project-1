# Decode Labs UI/UX Internship – Project 1

## Overview

This project was completed as part of the Decode Labs UI/UX Internship Program. The objective was to design a user-centered solution by applying UX principles, including research, wireframing, visual design, and prototyping.

## Objective

- Understand user requirements
- Create an intuitive interface
- Apply UX design principles
- Build a functional prototype in Figma

## My Role

- UX Research
- User Flow Design
- Wireframing
- UI Design
- Interactive Prototype

## Tools Used

- Figma

## Design Process

1. Understanding the problem
2. Defining user needs
3. Creating wireframes
4. Designing high-fidelity screens
5. Building the prototype

## Deliverables

- Wireframes
- High-fidelity UI
- Figma File
